Projects and applications using Cling

If your project or product is using Cling and you'd like to add it to this page, send an introduction to our Mailinglists.

  • BubbleDS

    A UPnP control point for Android and LinnDS streamer appliances.

  • BubbleUPnP

    A generic UPnP/DLNA audio control point and renderer for Android.

  • Jinzora UPnP

    The Jinzora music management and streaming server can be accessed through a UPnP MediaServer gateway written with Cling. The Android client also uses Cling to access the MediaServer.
